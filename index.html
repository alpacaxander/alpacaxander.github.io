Overview

The goal of this project is to practice creating a good foundation for a project without necessarily creating the project itself. This includes: infrastructure, deployment, developer environments, automated testing, loging, continuous profiling CICD, barebones api, login / user system, authentication & authorization.

Languages and technologies: java / kotlin, javascript / typescript, angular / react (tbd), maven, docker, kubernetes, elide, psql, liquibase, terraform, datadog.
